# Vehicle Maintenance System using Python

The Vehicle Maintenance System is a console-based application developed using *Python* to manage vehicle service records efficiently. This system helps track vehicle details, maintenance history, and upcoming service schedules using fundamental programming concepts and data structures.

---

## Objective

- To maintain vehicle service and maintenance records
- To apply Python programming concepts in a real-world scenario
- To simplify tracking of vehicle maintenance schedules

---

## Features

- Add vehicle details
- Update vehicle maintenance records
- View maintenance history
- Search vehicle by registration number
- Display all vehicles
- Delete vehicle records

---

## Technologies Used

- *Programming Language:* Python 3
- *IDE:* VS Code / PyCharm
- *Platform:* Console-based application

---

## System Requirements

- Python 3.7 or above
- Any text editor or Python IDE

---
